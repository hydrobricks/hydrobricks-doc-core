Detailed Description

HBV-96 upper-zone runoff (Lindström et al., 1997).

Non-linear outflow of the upper response box: Q0 = k × UZ^(1 + alpha)

With alpha = 0 the storage behaves as a linear reservoir. The response factor k is expressed in [1/d] for alpha = 0 (more generally [mm^(-alpha)/d]).
